## Contributing Translations

We welcome translations to additional languages! To contribute:

1. Copy `WHITEPAPER_EN.md` as your base
2. Translate the content (keep code examples and YAML unchanged)
3. Name your file `WHITEPAPER_XX.md` where XX is the ISO 639-1 language code
4. Submit a pull request

Please maintain the same structure and formatting as the English version.
